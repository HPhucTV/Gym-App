# Exercise catalog provenance

The version 1 exercise catalog is derived from the public-domain [Free Exercise DB](https://github.com/yuhonas/free-exercise-db) maintained by yuhonas. The upstream repository is released under the [Unlicense](https://github.com/yuhonas/free-exercise-db/blob/main/LICENSE), which dedicates the work to the public domain.

- Retrieval date: 2026-06-29
- Curated records: exactly 64
- Upstream identifier: `sourceId` stores the matching Free Exercise DB JSON filename without the `.json` suffix.
- Project-authored entries: `sourceId` uses `project:<id>` when no sufficiently close upstream exercise exists, especially for walking, low-impact, or cardio variants.
- Localization: Vietnamese names and concise technique instructions are project translations and rewrites; they are not copied from the upstream English instructions.
- Media: upstream image paths and image files are intentionally omitted from version 1.

The subset was reviewed for common bodyweight, dumbbell, barbell, cable, machine, core, and cardio exercises. The application bundles only the normalized Vietnamese JSON catalog, not the full upstream dataset.
